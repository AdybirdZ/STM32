# Examples - 示例代码文件夹

这个文件夹用于存放独立的示例代码片段，帮助快速学习和参考。

## 📂 子文件夹说明

### Basic - 基础示例
存放基础的、单一功能的示例代码：
- GPIO输入输出
- 串口发送接收
- 定时器基础使用
- 中断配置
- ADC读取

### Advanced - 进阶示例
存放复杂的、综合性的示例代码：
- DMA传输
- 多外设协同
- RTOS应用
- 通信协议实现
- 优化技巧

## 🎯 示例代码 vs 项目

| 特性 | 示例代码 (Examples) | 项目 (Projects) |
|------|-------------------|-----------------|
| 目的 | 学习单个功能 | 实现完整应用 |
| 规模 | 小型、简洁 | 完整、系统 |
| 文件 | 单个或少量文件 | 完整项目结构 |
| 独立性 | 可直接复制使用 | 需要完整环境 |

## 📝 示例代码组织方式

### 方式一：单文件示例
```
Basic/
├── gpio_blink.c          # GPIO闪烁示例
├── uart_printf.c         # UART打印示例
├── timer_delay.c         # 定时器延时示例
└── adc_read.c            # ADC读取示例
```

### 方式二：主题文件夹
```
Basic/
├── GPIO/
│   ├── blink.c
│   ├── button_input.c
│   └── README.md
├── UART/
│   ├── printf.c
│   ├── receive_interrupt.c
│   └── README.md
└── ...
```

## 📋 示例代码模板

每个示例文件开头应包含注释说明：

```c
/**
 * @file    gpio_blink.c
 * @brief   GPIO控制LED闪烁示例
 * @author  Your Name
 * @date    2024-01-01
 * 
 * @description
 * 本示例演示如何使用GPIO控制LED闪烁
 * 
 * @hardware
 * - MCU: STM32F103C8T6
 * - LED: PC13
 * 
 * @note
 * 使用HAL库实现
 */

#include "main.h"

// 示例代码...
```

## 💡 编写示例代码的建议

1. **简洁明了**：代码尽量简单，注释清晰
2. **可运行**：确保示例可以直接使用
3. **功能单一**：每个示例只演示一个知识点
4. **添加注释**：说明关键步骤和参数含义
5. **硬件说明**：标注使用的引脚和外设
6. **测试验证**：确保代码测试通过

## 📚 示例代码使用方法

1. **浏览示例**：查找需要的功能示例
2. **理解代码**：阅读注释，理解实现原理
3. **复制使用**：复制代码到自己的项目
4. **适配修改**：根据实际硬件调整参数
5. **测试验证**：测试功能是否正常

## 🔗 与其他文件夹的关系

```
Examples/        → 简单的示例代码，快速学习
    ↓
Projects/        → 完整的项目，综合应用
    ↓
Libraries/       → 提取的可复用库
```

## ⚙️ 快速创建示例

```bash
# 创建GPIO示例
mkdir -p Basic/GPIO
cd Basic/GPIO
touch blink.c README.md

# 创建UART示例
mkdir -p Basic/UART
cd Basic/UART
touch printf.c README.md
```

开始创建你的示例代码库吧！💻
